### Información general

Este proyecto nace a partir de la motivación de entregar a los clientes de una peluquería
una nueva forma de interactuar con el proveedor del servicio. La aplicación permite que el
cliente pueda revisar con detención todas las posibilidades de horario que tiene para agendar
un servicio de acuerdo a sus necesidades, considerando la disponibilidad del peluquero.
