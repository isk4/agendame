# Agéndame

Link al proyecto: [www.miriamdiban.com](https://www.miriamdiban.com/)

### Información general

Este proyecto nace a partir de la motivación de desarrollar una herramienta de administración
de agenda acotada, replicable y altamente personalizable para cualquier profesional independiente
que ofrezca servicios con toma de horario, entregando además ciertas estadísticas sobre el flujo
de clientes y los servicios que han consumido.

En este caso la aplicación fue desarrollada para una estilista: posee registro de clientes, notificaciones
vía correo electrónico (para confirmación de registro y hora tomada), muestra al cliente el estado actual
de la agenda del profesional en un calendario al momento de agendar una cita y permite consultar horarios
disponibles considerando la disponibilidad de la estilista y duración del servicio.

El dueño de la aplicación puede acceder a un panel de administración para hacer operaciones por lote con
las citas, consultar datos históricos, eliminar citas, etc. Este panel también mustra información graficada
sobre el flujo de clientes por día de la semana y cuáles son los servicios con mayor demanda.
